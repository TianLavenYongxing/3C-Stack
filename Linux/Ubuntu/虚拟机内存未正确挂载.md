### 正确扩展文件系统的方法
先确认文件系统类型
```
df -Th
```
示例输出可能是：
```text
Filesystem                         Type  Size  Used Avail Use% Mounted on
/dev/mapper/ubuntu--vg-ubuntu--lv  ext4   24G   24G     0 100% /
```
### 验证可用空间
```bash
vgdisplay
```
### 扩展LV
```bash
sudo lvextend -l +100%FREE /dev/mapper/ubuntu--vg-ubuntu--lv
```

### 扩展文件系统
```bash
sudo resize2fs /dev/mapper/ubuntu--vg-ubuntu--lv
```
验证结果
```bash
df -h
```
